快速開始使用

跟著步驟操作，三分鐘內完成安裝

下載最新版（GitHub Releases）

檔案名稱：inboxguardian-extension-v1.1.0.zip

1

下載並解壓縮

點擊上方按鈕下載 zip，解壓縮到固定位置（例如桌面或文件夾）。

⚠️ 別刪除這個資料夾，否則擴充功能會失效。建議放在一個不會亂動的地方。

2

打開 Chrome 擴充功能頁面

在 Chrome 網址列輸入以下指令並按 Enter：

chrome://extensions
3

開啟開發人員模式

頁面右上角切換「開發人員模式」開關。此為載入未上架擴充功能的標準做法，安全無虞。

4

載入擴充功能

點擊「載入未封裝項目」按鈕，選擇步驟 1 解壓縮的資料夾（裡面要有 manifest.json）。

5

打開 Gmail 即可使用

訪問 mail.google.com，InboxGuardian 會自動啟動並開始掃描未讀信件。

想使用 AI 增強模式？點擊瀏覽器右上角的 InboxGuardian 圖示，設定 API Key 即可。

常見問題

FAQ

有任何疑問？這裡可能有答案

InboxGuardian 會讀取我的郵件內容嗎？ +
不會讀取完整郵件內容。InboxGuardian 僅讀取 Gmail 介面上已顯示的寄件人、主旨、預覽片段（前 1500 字元）。不讀取附件、通訊錄或任何其他資料。所有偵測在你的瀏覽器本機執行。
一定要設定 API Key 嗎？ +
不需要。預設使用「快速模式」（純規則引擎），完全離線、免費、無需任何設定。如果你想要更高的偵測準確度，可以切換到「AI 增強模式」並設定 Claude、ChatGPT 或 Gemini 的 API Key（三選一）。
AI 模式的費用大概多少？ +

AI 只會在「規則引擎判定可疑」時才被呼叫，非常省錢。以一般使用者一天收到 50 封信、其中 1-2 封可疑為例：

  • • Gemini 2.5 Flash：每月約 NT$5 以下（甚至免費額度就夠）
  • • GPT-4o mini：每月約 NT$10
  • • Claude Haiku 4.5：每月約 NT$15
為什麼要開啟「開發人員模式」？是否安全？ +
因為 InboxGuardian 目前尚未上架 Chrome Web Store（付費驗證中），需要透過「載入未封裝項目」的方式安裝，這是 Chrome 官方提供的開發者標準流程。InboxGuardian 為開源軟體，你可以到 GitHub 檢視完整程式碼。後續會上架 Chrome Web Store，屆時可一鍵安裝。
支援哪些瀏覽器？ +
目前支援 Google Chrome、Microsoft Edge、Brave、Arc 等基於 Chromium 的瀏覽器。Firefox 與 Safari 版本規劃中。
我可以自訂規則或白名單嗎？ +
目前 v1.1 尚未提供圖形化的自訂介面，但因為是開源專案，進階使用者可直接修改 src/lib/rule-engine.js 新增白名單。圖形化自訂 UI 介面規劃於 v1.2 推出。
會不會誤判正常信件？ +

v1.1 大幅強化了誤判防護。InboxGuardian 採用「負面特徵加分 + 正向特徵減分」的平衡設計：

  • ✅ 合法行銷信有「退訂連結、廣告標示、推廣用語、指向信任網域」會自動減分
  • ✅ 釣魚信有「顯示名稱偽裝、帳戶警示、要求密碼」會加分
  • ✅ 邊緣案例交由 AI 智能判讀，辨識真實意圖

舉例：券商委託 SendGrid 發的申購廣告、國泰委託 EDM 平台發的信用卡優惠等常見誤判情境，v1.1 已能正確辨識為合法行銷信。如遇誤判請到 GitHub Issues 回報。

掃描會拖慢我的 Gmail 嗎？ +

幾乎感受不到。 v1.1 採用並發架構：

  • • 規則模式：同時處理 10 封信，10 封約 400ms 完成
  • • AI 模式：同時處理 3 封信（避免 API rate limit），20 封可疑信約 8 秒

掃描過程中你可以正常讀信、切資料夾，右下角會有浮動指示器顯示即時進度。完成後會自動淡化不打擾你。

如何更新到最新版？ +
到 GitHub Releases 下載最新版 zip → 解壓縮覆蓋原資料夾 → 到 chrome://extensions 點擊 InboxGuardian 的「重新載入」按鈕即可。
